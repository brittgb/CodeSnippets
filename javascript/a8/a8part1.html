What event should you almost always use to handle form submissions and why?

You should almost always add a submit handler to the form itself, which is triggered when the user clicks on the submit button OR when the user presses Enter/Return (meaning it is the most reliable/accessible approach)

Why should you always use a submit button (or image)?

Because otherwise, users without JavaScript can't submit the form.

What property do you use to fetch or set the value of a text input or text area?

value

How do you create linked select menus?

Add a change event to the first select menu, in the event handler change the options of the second menu based upon the selected value in the first.

How do you confirm that a checkbox was checked?

The state of any checkbox can be found by looking at its checked property, which will either be true or false using:

What JavaScript functions can be used to test a regular expression?

Test(), exec(), search(), match()

How do you match a literal character or string of characters?

To find a literal character or string of characters within other words, you don't need to use any meta characters. Or...use parenthesis to compare strings...in other words (abc) will match abc; or use indexOf() method to match an exact string.

What are meta-characters? How do you escape a meta-character?

Meta characters are special symbols that have a meaning beyond their literal value. To match any meta character, you will need to escape it with a backslash.

What meta-character do you use to bind a pattern to the beginning of a string? To the end?

^ binds a pattern to the beginning, and $ to the end.

How do you create sub patterns (aka groupings)?

Parenthesis are used to group characters into more involved patterns

What are the quantifiers? How do you require 0 or 1 of a character or string? 0 or more? 1 or more? Precisely X occurrences? A range of occurrences? A minimum of occurrences?

Quantifiers allow you to control the number of occurrences. 0 or 1 = ?, 0 or more = *, 1 or more = +, a precise amount is {x}, a range of occurrences is {x,y}, and a minimum of occurrences = {x,}

What are character classes?

Character classes are created by placing characters within square brackets, allowing you to match any one of a number of specified characters. You can indicate a range using a dash, for example [a-z].

What meta-characters still have meaning within character classes?

\, ^, -, ]

What shortcut represents the “any digit” character class? The “any white space” class? “Any word”? What shortcuts represent the opposite of these?

Any digit = \d

any white space = \s

any word = \w

opposites= \D, \S, \W
