How do you declare a variable? A variable is declared by using the var keyword. For example, you could say:
var fullName;
or...
var fullName = 'Brittany Barcroft';

What is variable scope? A variable's scope is the range in which it exists. In JavaScript, variables can be declare at a function-level or globally. Any variable that is used without first declaring it with var will automatically become a global variable, which means it applies to all functions within a program (and can cause errors).

What are the rules for a viarable's name? A variable name should start with a letter, underscore, or $. The rest of the name can be a combination of letters, numbers, and underscores. Names are case sensitive.

What is the assignment operator? The assignment operator is =, which is used to assign a value to a variable. For example:
var score = 95;
If you would like to indicate equality, you can do so by using == or !=. In JavaScript and other languages, a single equal sign (=) does not mean two things are equivalent.

What simple types were introduced in this chapter? In this chapter we were introduced to numbers, strings (a sequence of characters), and booleans (true/false).

How can you use a single quotation mark within a string? A double quotation mark? If your string is quoted with single quotations marks, you can use double quotes within the string and be ok. The reverse is also true. However, if your string is contained within double quotes and you'd also like to use double quotes within the string, you can use an escape sequence by prefacing each quotation mark within the string with a backslash. For example:

What does the *= operator do? How about +=? (There are two answers to this last question.) And what about ++? *= is an assignment operator, += is also an assignment operator plus the concatination operator for strings, and ++ is the increment operator. For *=, x*=y is the same thing as x=x*y and for +=, x+=y is the same thing as x=x+y. The increment operator (++) adds one to the value of a variable.

What operator can cause bugs when used with a string and a number together? The assignment operator can cause bugs because it is also the concatentation operator, and instead of adding two numbers you might accidentally concatenate something onto a number.

What does the toFixed() method do? The toFixed() method returns a number with the amount of places to the right of the decimal that you specify. For example:
var num = 4095.3892;
num.toFixed(3); // 4095.389
One thing to keep in mind is that toFixed() only formats the specified number but does not change the variable to the rounded number. If you are going to use the variable later on it will still be assigned to the original unrounded number.

Math is a global object for JavaScript, so it is always available for your use. Math uses predefined constants instead of variables.

What are escape sequences? Escape sequences allow you to parse strings literals for special characters/formatting. This includes the backslash escape to be used with quotations within a string, as well as \n for a new line, \r for a carriage return, or \\ for a literal backslash.
