GET is the standard method, and the type of request a browser makes when you click on a link. GET requests are best used to fetch data, and POST requests are intended to cause a server change or reaction.

What is caching and how does it affect Ajax processes?

Caching is used to improve performance by downloading resources whenever possible. This includes everything from images to CSS to JS to HTML. This means that the browser will attempt to cache the results of the Ajax request, which means changes made to the server-side resource won't immediately be reflected and make debugging harder.
