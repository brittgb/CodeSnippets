Unix Commands List:

  1. Pwd - displays the name of the directory you are currently in
  2. ls - lists the branches of the directory you are currently in
  3. cd - moves you to a different directory of your choosing
  4. mkdir - makes a new directory in the current diretory you are in
  5. pico
    1. one of the text editors for unix
    2. command used to create a text file of your choosing
